Clone the project

```bash
  git clone https://github.com/NAEEMGHADAI/tailwind-dashboard
```

Go to the project directory

```bash
  cd tailwind-dashboard
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm start
```
